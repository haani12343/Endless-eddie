# Endless Eddie

Welcome to **Endless Eddie**, a fast-paced endless runner where your goal is simple: survive as long as possible, collect coins, and beat your high score!

Take control of Eddie as he races through an ever-changing world filled with obstacles, props, and challenges. The longer you survive, the harder the game becomes, testing your reflexes and concentration. Dodge obstacles, stay on the path, and keep pushing forward to achieve the highest score possible.

## Features

* Endless runner gameplay with increasing difficulty
* Coin collection system
* High score tracking and saving
* Cartoon-style visuals and environment
* Dynamic obstacle spawning
* Main menu with animated camera and music
* Game over screen with score statistics
* Restart, Main Menu, and Quit options
* Sound effects and background music

## How to Play

* Move and avoid obstacles.
* Collect coins while running.
* Survive for as long as possible.
* Try to beat your previous high score.

## About the Project

Endless Eddie was developed using Unity as a personal game development project. The goal was to create a fun and polished endless runner experience while learning core game development concepts such as UI systems, scene management, audio, score tracking, obstacle spawning, game states, and player progression.

## Challenge Yourself

Every run is different. The longer you survive, the faster and more challenging the game becomes. Can you stay focused, dodge every obstacle, and set a new record?

Good luck, and have fun playing Endless Eddie!
